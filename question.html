What's the difference between let, const, and var in JavaScript?

I'm confused about variable declarations in JavaScript. When should I use each of these, and what are the key differences in scoping and hoisting?

Tags: JavaScript ES6

3 Answers

Correct!

The main differences are:
1. var is function-scoped and hoisted
2. let is block-scoped and can be reassigned
3. const is block-scoped but cannot be reassigned after declaration
Always prefer const by default, use let when you need to reassign, and avoid var in modern code.
